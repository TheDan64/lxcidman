# LXC ID Manager

lxcidman is a tool for managing unprivileged lxc containers' gid and uid mappings.
The goal is to have one centralized tool for making sure there are no overlapping mappings and
being able to fix them as well.

Note that it is very much still in development and may break your build. Please backup any important
configs before trying it out. We do not hold any responsibility for broken builds as a result of using
this tool.
